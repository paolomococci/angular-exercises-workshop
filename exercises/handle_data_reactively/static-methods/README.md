# StaticMethods

## scaffolding

```shell
ng n static-methods --routing --skip-tests --skip-install --style=sass
```

## install

```shell
npm install
```

## generate movie interface

```shell
ng g i model/movie --type=model
```

## generate data class

```shell
ng g cl repo/data --skip-tests --type=repo
```

## serve in development mode

```shell
ng serve-o
```
