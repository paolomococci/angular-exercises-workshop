# SettingUp

## scaffolding

```shell
ng n setting-up --routing --skip-tests --skip-install --style=sass
```

## installation

```shell
npm install
```

## install Jest

```shell
npm i jest --save-dev
npm i @types/jest --save-dev
npm i jest-preset-angular --save-dev --legacy-peer-deps
```

## serve in development mode

```shell
ng serve -o
```
