# table

Angular 14.1.2

## scaffolding

```shell
ng new table -S -g --routing --style=sass --skip-install
```

## install

```shell
npm install
```

## I install Angular Material to this template

```shell
ng add @angular/material
```

```shell
ng g c components/main
```
